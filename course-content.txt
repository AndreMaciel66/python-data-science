# Curso de Python para Data Science

Neste curso vamos aprender como dar os primeiros passos em Python e suas principais bibliotecas para coletar, transformar, salvar e visualizar estes dados.

Neste curso vamos ver...
- O que é Python?
- Como instalar e gerenciar seus ambientes virtuais.
- Anaconda ou VirtualEnv?
- Gerenciando suas bibliotecas
- NumPy and Pandas
- Matplotlib


Python is a programming language that lets you work quickly and integrate systems more effectively.
